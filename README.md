DeviceRent — это серверная часть веб-приложения для аренды устройств.
Проект реализован на ASP.NET Core 8 с использованием:
    Entity Framework Core
    SQL Server
    JWT-аутентификации
    REST API (контроллеры)
    Middleware для обработки исключений
    CORS-политики
API позволяет:
    Регистрировать и авторизовывать пользователей
    Работать с заказами (создание, обновление, удаление)
    Загружать аватар пользователя
    Оставлять отзывы
Используемые технологии:
    C#
    HTML
    CSS
    ASP.NET Core 8
    Entity Framework Core
    SQL Server
    JWT Authentication
    REST API
    Dependency Injection
    Middleware
    CORS
